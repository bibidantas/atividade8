# 📊 Pesquisa de Satisfação do Cliente — Tudoweb

Um projeto em ![Python](https://img.shields.io/badge/Python-3.10-blue?logo=python&logoColor=white) focado em coleta e análise de feedback de clientes, ideal para treinar lógica de programação, estruturas de repetição e tomada de decisão.

## 🌐 Visão Geral

Este programa simula uma pesquisa de satisfação com até 50 entrevistados, permitindo entender como os clientes avaliam o atendimento de uma plataforma fictícia chamada Tudoweb.

O sistema coleta dados básicos e gera um resumo final das avaliações.

## 🛠️ Tecnologias Utilizadas:  <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/python/python-original.svg" width="40"/>


## 🎯 Objetivo
- Praticar conceitos fundamentais de programação em Python
- Simular um cenário real de coleta de feedback
- Organizar e interpretar dados simples

## ⚙️ Funcionamento Detalhado
### 1. Repetição (Loop)

O programa utiliza um for para entrevistar 50 pessoas:

```for i in range(1, 51)```:  
### 2. Coleta de Dados

Para cada usuário:

Nome
Idade  
Avaliação do atendimento  
```
nome = input("Digite seu nome: ")
idade = int(input("Digite sua idade: "))
```
### 3. Avaliação

O usuário escolhe uma opção:

Código	Avaliação  
1	Excelente  
2	Bom  
3	Ruim  

### 4. Decisão (Condicionais)

O programa usa ```if/elif/else``` para:

- Identificar a avaliação  
- Exibir mensagem personalizada  
- Atualizar contadores  

### 5. Saída do Programa

Ao final, o sistema mostra:
```
===== Resultado da Pesquisa de Satisfação =====  
Total de avaliações EXCELENTE: X  
Total de avaliações RUIM: Y
```
